# → Wats⚉n Converstion Chat ←

Simples aplicação de conversação com Inteligência Artificial: **Watson**

### Instalação

Watson requires [Node.js](https://nodejs.org/) v4+ to run.

Instale as dependeências e start server.

```sh
$ cd watsonbot/
$ npm install 
$ node app.js
```
